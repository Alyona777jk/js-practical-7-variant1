# Практичні завдання з JavaScript (Тема 1-2)

Цей репозиторій містить розв'язання практичних завдань з основ JavaScript, що охоплюють роботу зі змінними, циклами, функціями, об'єктами та маніпуляціями з DOM.

## 📋 Опис завдань

### Завдання 1: FizzBuzz
Функція `fizzBuzz(n)`, яка генерує масив рядків від 1 до `n` із заміною чисел, кратних 3 та 5, на відповідні слова:
- "Fizz" (ділиться на 3)
- "Buzz" (ділиться на 5)
- "FizzBuzz" (ділиться на обидва)

### Завдання 2: Об'єкт Студента
Функція `createStudent(name, grades)`, яка обчислює середній бал та визначає статус успішності (прохідний бал >= 60).

### Завдання 3: Перевірка на паліндром
Функція `isPalindrome(str)`, яка перевіряє, чи є рядок паліндромом, ігноруючи регістр, пробіли та розділові знаки.

### Завдання 4: Інтерактивний список справ (DOM)
Реалізація To-Do списку з можливостями:
- Додавання нових елементів через `input`.
- Видалення елементів при кліку (використано делегування подій).
- Очищення поля введення після успішного додавання.
